# Automated Inventory & Nutritional Intelligence

## Description
A Streamlit-based AI application that classifies fruits and vegetables using a VGG deep learning model and displays shelf-life and nutritional information.

## Features
- Image classification using VGG
- Fruit/Vegetable detection
- Confidence score
- Shelf-life prediction
- Nutritional information
- Interactive Streamlit interface

## Technologies Used
- Python
- Streamlit
- TensorFlow
- Keras
- NumPy
- Pillow
- JSON

## Installation

pip install -r requirements.txt

streamlit run app.py
